# SIVET - Sistema de Información Veterinaria 🐾

**SIVET** es una solución tecnológica diseñada para la **Veterinaria Máscol-Colombia**, enfocada en la digitalización y optimización del proceso de agendamiento de citas y gestión operativa.

---

## 📝 Planteamiento del Problema
Las clínicas veterinarias enfrentan dificultades constantes en la gestión manual de citas y registros. El uso de agendas físicas y llamadas telefónicas genera:
* **Errores humanos** y duplicidad de citas.
* **Pérdida de información** crítica de pacientes.
* **Sobrecarga operativa** para el personal administrativo.
* **Baja satisfacción del cliente** debido a tiempos de respuesta elevados.

### Pregunta Problema
> ¿Cómo el desarrollo de una aplicación web para el agendamiento de citas puede optimizar la gestión operativa y mejorar la atención al cliente en la veterinaria Máscol-Colombia?

---

## 🎯 Objetivos del Proyecto

### Objetivo General
Desarrollar un sistema de información integral para la empresa **Máscol-Colombia** que permita la gestión eficiente del agendamiento de citas y servicios veterinarios.

### Objetivos Específicos
1. **Interfaz Intuitiva:** Diseñar un entorno ágil para que el personal administrativo registre y gestione citas sin complicaciones.
2. **Validación en Tiempo Real:** Implementar un módulo que elimine cruces de horarios y errores de asignación.
3. **Base de Datos Centralizada:** Garantizar el almacenamiento seguro de información de clientes, mascotas y personal médico.
4. **Automatización de Agenda:** Organizar los flujos de atención según la disponibilidad real del personal.
5. **Arquitectura Escalable:** Construir un sistema preparado para futuras integraciones y mejoras funcionales.

---

## 🚀 Alcance y Delimitación

El proyecto busca sustituir los registros manuales por una "única fuente de verdad" digital, accesible para recepcionistas y médicos veterinarios.

### Funcionalidades Principales
* **Gestión de Citas:** Organización y asignación de horarios inteligente.
* **Registro de Clientes y Mascotas:** Base de datos digital completa.
* **Módulo de Control:** Confirmación y cancelación de servicios.
* **Gestión de Personal:** Control de acceso y roles para empleados.

### Áreas del Proyecto
| ✅ Áreas Incluidas | ❌ Áreas Fuera de Alcance |
| :--- | :--- |
| Recepción y Administración | Contabilidad |
| Cuerpo Médico Veterinario | Tesorería |

---

## 💡 Justificación
La implementación de **SIVET** responde a una necesidad estratégica de modernización. Al centralizar la información, se garantiza la integridad de los datos y se mejora la capacidad de respuesta. 

**Beneficios clave:**
* **Para los empleados:** Reducción de carga operativa y procesos manuales.
* **Para los clientes:** Servicio más ágil, profesional y confiable.
* **Para la dirección:** Control total del negocio bajo un modelo moderno y escalable.

---

## 🛠️ Desarrollo y Escalabilidad
Este proyecto se desarrolla en un marco académico (ADSO) con una duración estimada de **un año y medio**. La arquitectura está pensada para evolucionar, permitiendo que el software se adapte a nuevos requerimientos del sector veterinario en el futuro.
